CLASS: Customer
Description: Represents individuals who hold accounts in the bank, identified by attributes such as name and address.
ATTRIBUTES:
- name: String
- address: String
- customerID: String
METHODS:
- getName(): String
- getAddress(): String
- updateAddress(newAddress: String): void
- getCustomerID(): String

CLASS: Account
Description: A general representation of a financial account held by a customer, with attributes like balance and account type.
ATTRIBUTES:
- balance: Double
- accountNumber: String
- accountType: String
METHODS:
- getBalance(): Double
- deposit(amount: Double): void
- withdraw(amount: Double): void
- getAccountNumber(): String
- getAccountType(): String

CLASS: SavingsAccount
Description: A specific type of account that offers an interest rate, inheriting from the Account class.
ATTRIBUTES:
- interestRate: Double
METHODS:
- calculateInterest(): Double
- applyInterest(): void

CLASS: InvestmentAccount
Description: A specific type of account used for buying stocks, inheriting from the Account class.
ATTRIBUTES:
- stocks: List<Stock>
METHODS:
- buyStock(stock: Stock, quantity: int): void
- sellStock(stock: Stock, quantity: int): void
- getStocks(): List<Stock>

CLASS: Stock
Description: Represents the financial instruments that can be bought and sold within investment accounts, including attributes like quantity and price (ticker).
ATTRIBUTES:
- ticker: String
- quantity: int
- price: Double
METHODS:
- getTicker(): String
- getQuantity(): int
- getPrice(): Double
- updatePrice(newPrice: Double): void

CLASS: Commission
Description: Represents the fee applied by the bank on stock orders, affecting transactions made in investment accounts.
ATTRIBUTES:
- rate: Double
METHODS:
- calculateCommission(amount: Double): Double
- getRate(): Double

RELATIONSHIPS:
- Customer -- Association -- Account 1..*
- Account -- Inheritance -- SavingsAccount 1
- Account -- Inheritance -- InvestmentAccount 1
- InvestmentAccount -- Aggregation -- Stock 0..*
- Commission -- Association -- InvestmentAccount 0..*
- Stock -- Association -- Commission 0..*
